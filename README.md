【游戏名称】雪人游戏


【游戏介绍】
这是一款H5原生开发的趣味休闲类小游戏。玩家通过鼠标操控雪人，接取正确的雪人身体部分来完成雪人构建，同时需要躲避带有"×"的图标，以获得高分并完成游戏目标。


【玩法说明】
操作：
- 使用鼠标移动操控雪人左右移动
- 点击鼠标左键或按住鼠标拖动调整雪人的位置
- 观察下落的物品，及时调整雪人位置接取或躲避


道具：
- 雪人头：接取后增加头部部分，获得基础分数
- 雪人身：接取后增加身体部分，获得中等分数
- 雪人手臂：接取后增加手臂部分，获得中等分数
- 雪人纽扣：接取后增加装饰部分，获得额外分数
- 带有"×"的图标：躲避此类图标，接触后会减少分数或失去已获得的部分


规则：
- 物品从屏幕顶部随机位置下落
- 玩家需要操控雪人左右移动接取正确的物品
- 接取到带有"×"的图标会导致扣分或失去部分已获得的雪人部件
- 当成功接取到所有必要的雪人身体部分后，可以完成一个完整的雪人构建
- 游戏有时间限制或分数目标，达成后进入下一关卡
- 随着关卡提升，物品下落速度加快，游戏难度增加


目标：
- 接取正确的雪人身体部分，完成雪人构建
- 躲避带有"×"的图标，避免扣分或失去部件
- 获得尽可能高的分数
- 完成所有关卡的雪人构建任务
- 挑战自己的反应速度和精准操控能力


Tips：
- 保持雪人的移动平稳，避免过于急促的操作导致错过物品
- 优先接取大型的雪人部件（如头、身体），再收集小型装饰
- 观察物品下落的轨迹和速度，提前预判并调整位置
- 注意屏幕底部的时间或分数显示，合理安排策略
- 当多个物品同时下落时，优先接取重要部件，放弃次要物品
- 随着关卡提升，适当降低移动速度，提高接取准确率


技术介绍：
- 使用HTML5 Canvas和原生JavaScript开发
- 实现了流畅的雪人移动和物品下落动画效果
- 采用碰撞检测算法判断物品接取和躲避情况
- 支持PC端鼠标操控和移动端触摸操作
- 包含动态难度调整、得分系统和关卡进阶机制
- 优化了游戏性能，确保在不同设备上都能流畅运行
